ỨNG DỤNG DANH SÁCH LIÊN KẾT ĐƠN
Chương trình xét học bổng cho sinh viên

 Sinh viên thực hiện
 Họ tên: Huỳnh Minh Khang

 Giảng viên hướng dẫn
 Họ tên: Lê Minh Tự 

Mô tả đề tài
Đề tài xây dựng chương trình quản lý danh sách sinh viên bằng cấu trúc
dữ liệu danh sách liên kết đơn, từ đó áp dụng để xét học bổng cho sinh viên
dựa trên các tiêu chí như điểm trung bình.

 Mục tiêu
- Tìm hiểu danh sách liên kết đơn
- Áp dụng vào bài toán xét học bổng
- Rèn luyện tư duy thuật toán
- Thực hành lập trình C/C++

 Chức năng chính
- Thêm sinh viên
- Xóa sinh viên
- Sửa thông tin sinh viên
- Hiển thị danh sách sinh viên
- Xét sinh viên đủ điều kiện học bổng

 Công nghệ sử dụng
- Ngôn ngữ: C / C++
- Môi trường: Dev-C++
